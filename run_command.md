## How to Run the Stand alone main.py,
## from root folder run the following
>python -m server.main
>
## RUN THE restAPI server app.py
## From root folder run the following command
>uvicorn server.app:app --reload --port 8000

## RUN the Client(streamlit ui client)
## from the root folder run the following
>streamlit run client/streamlit_ui_app.py
